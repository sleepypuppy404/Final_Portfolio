Getting Started




For this project, I did extensive research on 15 different breeds of dog. Although I know breeding is an imperfect system at best, I could not think about different ways to catagorize dogs than by breeds. Thus, I collected data on each dog breed, and compiled it into a dataset for use in the project

Planned Features


In order to make this as effective as possible, in addition to simply displaying a breed of dog's information, I wanted to make it easy to display and see quick information. As a result, I added several different filtering and sorting methods, in order to allow users to narrow down what sort of pet they want. For users adopting multiple dogs at once, I wanted to make some sort of aggregate screen that would allow them to get a sense for the general requirements for their new dogs. These requirements are included below:

Filtering:

  • Breed Type
  • Price

Sorting:

  • Friendliness
  • Activity Level
  • Price

Aggregator:

  • Avg Friendliness
  • Avg Activity per day
  • Approximate Cost

Takeaways

Lessons Learnt

  • Proper preparation work makes coding the website easier
  • Color scheme can make or break the design of a website
  • A visual design guide makes prototyping more organized

Insights

  • Brown works really nicely with lighter colors
  • Cafes as a whole either use a light fall asthetic or a darker green one
  • Sidemenus can be really tricky to implement as a whole.
